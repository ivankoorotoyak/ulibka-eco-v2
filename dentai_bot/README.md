# dentai_bot

## Описание
Бот экосистемы "Улыбка" - dentai

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start dentai_bot
```

## Логи
```bash
tail -f /var/log/smile_ecosystem/dentai_bot.log
```
