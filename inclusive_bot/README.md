# inclusive_bot

## Описание
Бот экосистемы "Улыбка" - inclusive

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start inclusive_bot
```

## Логи

tail -f /var/log/smile_ecosystem/inclusive_bot.log
