# dream_bot

## Описание
Бот экосистемы "Улыбка" - dream

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start dream_bot
```

## Логи

tail -f /var/log/smile_ecosystem/dream_bot.log
