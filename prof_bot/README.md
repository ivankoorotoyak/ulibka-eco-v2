# prof_bot

## Описание
Бот экосистемы "Улыбка" - prof

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start prof_bot
```

tail -f /var/log/smile_ecosystem/prof_bot.log
