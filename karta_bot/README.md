# karta_bot

## Описание
Бот экосистемы "Улыбка" - karta

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start karta_bot
```

## Логи
```bash
tail -f /var/log/smile_ecosystem/karta_bot.log
```
