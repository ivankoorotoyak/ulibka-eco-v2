# implant_bot

## Описание
Бот экосистемы "Улыбка" - implant

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

```bash
systemctl start implant_bot
```
