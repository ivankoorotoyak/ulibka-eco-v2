# philo_bot

## Описание
Бот экосистемы "Улыбка" - philo

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start philo_bot
```

## Логи
```bash
tail -f /var/log/smile_ecosystem/philo_bot.log
```
