# admin_bot

## Описание
Бот экосистемы "Улыбка" - admin

## Команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Помощь |
| `/info` | Информация |

## Запуск
```bash
systemctl start admin_bot
```

## Логи
```bash
tail -f /var/log/smile_ecosystem/admin_bot.log
```
